## Project Setup
### To start the project follow the instructions below 

```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
yarn test:unit
```

### Alternative way to run using docker

```sh
cd 'path-to-project'
```

```sh
docker compose up -d --build
```

#### Or

```sh
docker-compose up -d --build
```
